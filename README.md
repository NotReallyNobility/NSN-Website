# NSN-Website
A website collecting details on my stories.

The current website uses Bootstrap5 and Jquery. 
The current bootstrap features in use are the containers, navbar, cards, and flex.
The current Jquery features in use are fade in and the animate function(grows the logo)

The current finished pages are the home page and the setting selection page (places, not configurations), though no specific setting pages are finished.

To accomplish item specific pages, javascript will be used to fill out templates as opposed to creating individual files. This will make the files smaller, but will require decent commenting to keep understandable.
